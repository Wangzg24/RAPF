This is our UBPF's first attempt in the field of one-shot image classification.

You need to create a new folder named "data" and download the official Mini-Imagenet dataset (A usable download link: https://mtl.yyliu.net/download/). The dataset should be placed in the following locations: data/mini_imagenet/train, data/mini_imagenet/eval, and data/mini_imagenet/test.
